User Flow

PRD §5.5
진입 → 언어선택 → 권한 → 세션 → 인식/안내 → 기록
진입 ─▶ [/ 랜딩·언어선택]
          │ 언어 선택 + 시작
          ▼
      {카메라·마이크 권한}
        ├─ 거부 ─▶ (session: no-permission)
        └─ 허용 ─▶ [/session 실시간 동행]
                     │ 서류 비춤 + 음성질문
                     ▼
                 {recognize_document}
                   ├─ confidence < 0.5 ─▶ 재촬영 유도(단정 금지, FR-014)
                   └─ 인식 성공 ─▶ 칸별 작성법 모국어 음성
                                    ├─ 행정용어 ─▶ lookup_admin_term (AI Hub grounding)
                                    ├─ 키오스크 ─▶ 버튼 위치 음성 안내(FR-010)
                                    └─ 종료 ─▶ create_record
                                                 ▼
                                        [/my 기록목록] ─▶ [/record/:id 상세]
